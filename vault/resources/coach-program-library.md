# Coach Program Library — Real Published Programs (Reference Index)
*Compiled July 2026 from public sources. Companion to `coach-methodologies-research.md` — that file covers each coach's *philosophy*; this file indexes their *actual published programs/templates* with structure, split, rep scheme and where to find them.*

> **Copyright / usage note.** These are third-party coaches' own published, copyrighted programs. This file documents structure, splits, rep schemes and parameters as reference — the way a review or database entry does — so VitalEdge/Pinnacle can understand how respected coaches build programs and design comparable (not copied) block templates. **Do not reproduce any of these programs wholesale into VitalEdge as first-party content or resell them.** Where a program is paid, the entry says so and links to the source; buy it there if you want the full session-by-session detail. Coach names stay out of client-facing VitalEdge UI per the existing methodology rule.

**Scope note on Paul Carter:** per instruction, only his *current* catalogue (2023–2026, TrainHeroic era) is indexed. His older blog-era templates (Base Building, Strong-15, LRB split, 2011–2013) are deliberately excluded.

## A. Kuba Cielen — "The Kuba Method" & published splits
*Platform: teammkcoaching.com / TrainHeroic-style delivery. Method hallmarks: 2 working sets to RIR 0–1 after 2–3 ramp sets, strict 3-1-1-0 tempo, MEV→MAV→MRV volume progression, reps-first double progression, 6-week mesocycle (3 accumulation / 2 intensification / 1 deload).*

**Kuba's 2025 high-frequency PPL — full split (publicly detailed on BarBend/Instagram; the most complete free Kuba template).**
Each body part every 4 days: 3 consecutive training days + 1 rest, repeating. Mon Push A · Tue Pull A · Wed Legs A · Thu rest · Fri Push B · Sat Pull B · Sun Legs B · Mon rest.

1. **Push A (16 sets — lighter, higher volume, delt-biased):** Machine Lateral Raise · Machine Pec Flye · Incline DB Press · Flat Machine Press · Deficit Push-Up (unmeasured) · Standing Triceps Press · Cable Overhead Extension · Single-Arm Cable Lateral Raise.
2. **Push B (16 sets — heavy tension, compound):** Lying Cuffed Cable Lateral Raise · Low-Incline DB Press · Incline Machine Chest Press · Dips (deep ROM) · Incline Cable Flye · Cable Overhead Press · Machine Lateral Raise.
3. **Pull A (17 sets — lat-biased):** Single-Arm Pulldown ×2 · Chest-Supported Row ×3 · Single-Arm Lat Row ×3 · Neutral-Grip Row ×3 · Shrugs ×1 · Cable Preacher Curl ×3 · Machine Rear Delt Flye ×2.
4. **Pull B (17 sets — 7 lat / 4 upper back):** Single-Arm Cable Row · T-Bar Row · Single-Arm Machine Pulldown · Seated Upper Back Row · Single-Arm Plate-Loaded Row · Bayesian Biceps Curl · Cable Reverse Flye.
5. **Legs A (11 sets — quad-focused):** Hamstring Curl ×3 · Leg Extension ×2 · Safety-Bar Squat ×1 · Leg Press · Lying Leg Curl · Calf Press. (Hamstrings first, à la Jay Cutler; SSB front-squat pattern for upright torso + knee flexion.)
6. **Legs B (11 sets — hip-hinge / hamstring stretch bias):** Hamstring Curl ×3 · Leg Extension ×3 · Squat Press ×1 · Deficit RDL · Calf Press.

**Method-based mesocycle templates (structure published; full plans via teammkcoaching / Kuba's app):**
7. **Offseason mass mesocycle** — 6-wk, full split, MEV→MRV volume ramp, reps-first progression on all lifts.
8. **Mini-cut mesocycle** — 6-wk, volume held at MAV, intensity techniques (myo-reps/rest-pause) introduced wks 4–5 to preserve stimulus in a deficit.
9. **Reverse-diet mesocycle** — 6-wk, gradual volume re-introduction from deload baseline as calories climb.
10. **Contest-prep mesocycle** — 6-wk, retention-focused, volume capped, higher-effort sets prioritised (matches the "don't inflate prep volume" doctrine in the methodology file).
11. **Chest specialization block** — MEV 8 / MAV 14 / MRV 20 landmarks, lengthened-position priority (incline/deficit work).
12. **Back specialization block** — MEV 10 / MAV 16 / MRV 22, lat-bias, heavy rows over pulldowns for mind-muscle connection.
13. **Quad-focus lower structure** — one hard quad session/week (deep-knee-flexion stretch limiter), lighter ham/calf top-up elsewhere (the A/B lower logic in the methodology file).
14. **Standard 4-day upper/lower** — the entry-level Kuba Method layout before the high-frequency PPL; 2 working sets/exercise, same tempo/progression rules.

*Where to get the full plans:* teammkcoaching.com/kuba · @kuba_sylvester_cielen · YouTube "Kuba Cielen" (free split breakdowns) · Arvo method guide (free write-up of the system).

## How to use this for VitalEdge (design guidance, not copying)

These programs validate several block-template shapes worth supporting in `promptBuilder.ts` / the block schema — build *comparable* structures, never copy a coach's exact program:
- **6-week mesocycle with 3/2/1 internal structure** (Kuba) maps cleanly onto VitalEdge's existing 6+1 block. His accumulation→intensification→deload is the reference for what "changes" week to week inside a block.
- **Every-4-day frequency with A/B day variants** (Kuba PPL, Carter 2-up/2-low) is a frequency pattern the app could offer as a template, with the effective-volume-falls-as-frequency-rises rule already documented in the methodology file.
- **1–3 exercises/muscle, 12–20 sets/muscle/week, lengthened+mid-range bias** (Bennett) is a clean set of default guardrails for AI-generated hypertrophy sessions — all logging-only-compatible (no load math).
- **Rest-pause / DC as an "intensification technique" tag** (Carter, Kuba wks 4–5) — a switch the methodology selector can flip for advanced `betaTester` clients, distinct from straight-set defaults.
- **Goal-phase splits** (offseason / mini-cut / prep for Kuba; bulk / cut / glute for Carter) validate having phase-tagged block templates rather than one generic block.

*All three coaches independently converge on: heavy-ish progressive overload as the primary driver, execution/ROM quality gating progression, and modest per-session volume repeated at higher frequency — which is exactly the logging-only, RIR-based, frequency-aware model VitalEdge is already built around.*
