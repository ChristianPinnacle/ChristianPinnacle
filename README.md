# Saiyan Archive

Personal AI second brain — markdown vault + interactive graph + RAG chat (coming in later phases).

## Quick start

```bash
npm install
npm run dev
```

- Frontend: http://localhost:5174
- Backend: http://localhost:3001

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start frontend + backend |
| `npm run test` | Run vitest tests |
| `npm run reindex` | Vault re-scan (stub until Task 2) |
| `npm run db:generate` | Generate Drizzle migrations |
| `npm run db:migrate` | Run Drizzle migrations |
